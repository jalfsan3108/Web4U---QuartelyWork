Introducción al CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para definir la presentación de un documento HTML o XML. Permite separar el contenido de la estructura y la apariencia, facilitando el diseño de páginas web.

Selección de Elementos

En CSS, puedes seleccionar elementos de HTML utilizando diferentes selectores. Los más comunes son:

  • Selector de Etiqueta: Selecciona todos los elementos de una etiqueta HTML. Ejemplo: p { color: red; }
  • Selector de Clase: Selecciona todos los elementos con una clase específica. Ejemplo: .mi-clase { font-size: 20px; }
  • Selector de ID: Selecciona un único elemento con un ID específico. Ejemplo: #mi-id { background-color: blue; }

Propiedades Comunes de CSS

Algunas de las propiedades más comunes en CSS incluyen:

  • color: Cambia el color del texto.
  • background-color: Define el color de fondo de un elemento.
  • font-size: Define el tamaño de la fuente.
  • margin: Establece el margen alrededor de un elemento.
  • padding: Establece el relleno dentro de un elemento.

Ejemplo Básico de CSS

Veamos un ejemplo sencillo de CSS aplicado a una página HTML:

body {
background-color: #f0f0f0;
}

h1 {
color: blue;
text-align: center;
}

p {
font-size: 18px;
color: #333;
}

Este código cambia el color de fondo del body, el color y la alineación del h1, y el tamaño y color del texto en los p.

Conclusión

CSS es esencial para crear páginas web visualmente atractivas. A medida que practiques, aprenderás a usar propiedades y técnicas más avanzadas para mejorar el diseño de tus sitios web.
