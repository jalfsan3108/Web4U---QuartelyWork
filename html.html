Bienvenido a la Guía Básica de HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Utiliza una serie de etiquetas que definen la estructura y el contenido de la página.

Estructura Básica de una Página HTML

Una página HTML básica consta de los siguientes elementos principales:

  • <!DOCTYPE html> - Define el tipo de documento.
  • <html> - Etiqueta raíz que envuelve todo el contenido.
  • <head> - Contiene metadatos como el título y enlaces a recursos.
  • <body> - Contiene el contenido visible de la página.

Ejemplo Básico de HTML

A continuación se muestra un ejemplo básico de una página HTML:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi Primera Página</title>
</head>
<body>
  <h1>Hola, Mundo</h1>
  <p>¡Bienvenido a tu primera página web!</p>
</body>
</html>

Etiquetas Comunes en HTML

  • <h1> - <h6>: Etiquetas de encabezado (de mayor a menor importancia).
  • <p>: Define un párrafo.
  • <a>: Crea un enlace.
  • <img>: Inserta una imagen.
  • <ul>: Define una lista sin orden.
  • <ol>: Define una lista ordenada.
  • <li>: Define un ítem de lista.

¿Qué Son los Atributos?

Los atributos son propiedades adicionales que se pueden agregar a las etiquetas HTML. Se encuentran dentro de la etiqueta de apertura. Por ejemplo:

<a href="https://www.example.com">Visita mi página</a>

En este caso, href es un atributo que define la URL de destino del enlace.
