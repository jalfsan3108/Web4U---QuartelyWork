Introducción al Lenguaje SVG

¿Qué es SVG?

SVG (Scalable Vector Graphics) es un formato de imagen vectorial que se utiliza para crear gráficos bidimensionales en la web. A diferencia de las imágenes rasterizadas, los gráficos SVG no pierden calidad al ser escalados, ya que están basados en vectores (líneas, formas y curvas). Es especialmente útil para la creación de logos, iconos y gráficos interactivos.

Elementos Básicos de SVG

Un archivo SVG está compuesto por elementos dentro de una etiqueta `

<svg width="200" height="200">
    <circle cx="100" cy="100" r="50" fill="red" />
    </svg>

Este código crea un círculo rojo en el centro de un lienzo SVG de 200x200 píxeles. Los atributos importantes son:

  • cx: Coordenada X del centro del círculo.
  • cy: Coordenada Y del centro del círculo.
  • r: Radio del círculo.
  • fill: Color de relleno del círculo.

Otros Elementos Comunes en SVG

Además de círculos, SVG permite crear otras figuras como rectángulos, líneas y polígono. Aquí tienes ejemplos básicos:

<svg width="200" height="200">
    <rect x="50" y="50" width="100" height="100" fill="blue" />
    <line x1="50" y1="50" x2="150" y2="150" stroke="green" stroke-width="2" />
    </svg>

Este código dibuja un rectángulo azul y una línea verde en un lienzo SVG de 200x200 píxeles. Los atributos para estos elementos son:

  • rect: Para crear un rectángulo, usamos los atributos x, y, width, height y fill.
  • line: Para crear una línea, usamos los atributos x1, y1, x2, y2, stroke (color de la línea) y stroke-width (grosor de la línea).

SVG y Estilos CSS

Al igual que los elementos HTML, los elementos SVG también pueden ser estilizados con CSS. Puedes cambiar colores, tamaños y otros atributos visuales.

<svg width="200" height="200">
    <circle cx="100" cy="100" r="50" class="my-circle" />
    </svg>

.my-circle {
    fill: yellow;
    stroke: black;
    stroke-width: 5;
}

En este ejemplo, el círculo se estiliza utilizando una clase de CSS llamada .my-circle que cambia su color de relleno a amarillo, agrega un borde negro y un grosor de 5 píxeles.
